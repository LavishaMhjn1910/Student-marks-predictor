# 🎓 Student Marks Predictor

## 📌 Project Overview
This project predicts a student's marks based on the number of hours studied using a Machine Learning Linear Regression model.

## 🚀 Features
- Predicts marks based on study hours
- Built with Python and Scikit-learn
- Interactive web app using Streamlit

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Streamlit
- Joblib

## 📂 Project Structure

student-marks-predictor/
│── app.py
│── student_marks_model.pkl
│── Student_Marks_Predictor.ipynb
│── requirements.txt
│── README.md

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📈 Machine Learning Model
- Algorithm: Linear Regression
